## React Recruitment

- `state managment` without using 3rd party libraries.

- reusable [`custom hooks`](./src/hooks).

- `Atomic design`.

- Mocked Service Worker [`msw`](https://mswjs.io/)

### Tech Stack

- TypeScript
- Next.js
- Emotion
- [`Storybook`](https://storybook.js.org)
- [`MSW`](https://mswjs.io/)

### start

```docker
# install dependencies
yarn

# start dev on port `3000`
yarn dev

# storybook
yarn storybook
```
